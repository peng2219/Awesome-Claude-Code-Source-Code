## 源码恢复

将 Claude Code 原始源码恢复为可读、可构建状态的项目。

[claude-code](https://github.com/claude-code-best/claude-code) - 反编译的 Claude Code 完整可构建版本，TypeScript 类型全修复，可直接运行和调试。作者 @claude-code-best。

## 重实现 & 替代方案

受 Claude Code 启发的开源重写与替代实现。

[claude-code](https://github.com/instructkr/claude-code) - 使用 Rust 全面重写的 Claude Code，目标是功能对等。作者 @instructkr。  
[open-agent-sdk](https://github.com/shipany-ai/open-agent-sdk) - 无 CLI 依赖的开源 Agent SDK，claude-agent-sdk 的替代方案。作者 @shipany-ai。  
[cc-mini](https://github.com/e10nMa2k/cc-mini) - 超轻量级 Python AI Agent Harness 脚手架，Claude Code 的迷你版。作者 @e10nMa2k。

## 研究 & 分析

关于 Claude Code 内部实现的深度研究、文档与书籍。

[claude-code-deep-dive](https://github.com/tvytlx/claude-code-deep-dive) - 基于 claude-code npm 包中 cli.js.map 源码恢复的系统性研究。作者 @tvytlx。  
[Claude Code Unpacked](https://ccunpacked.dev/) - Claude Code 内部机制全景映射：Agent 循环、50+ 工具、多 Agent 编排以及未发布特性。  
[Harness Books](https://github.com/wquguru/harness-books) - 关于 Harness 工程的两本书：《Claude Code 设计指南》和《Claude Code 和 Codex 的 Harness 设计哲学》。作者 @wquguru。
